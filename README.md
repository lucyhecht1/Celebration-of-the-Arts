# Yavneh Celebration of the Arts
A full-stack website built to support [Yavneh’s Celebration of the Arts](https://yavnehcelebrationofthearts.onrender.com/), a community-wide event that showcases the Columbia Jewish Orthodox community's talent in music, visual art, and performance. The website handles event information, RSVPs, and a gallery of past events, serving 200+ users and facilitating an event with 150 attendees.

---

## Features
* **Homepage** Landing page highlighting the event with clear navigation to RSVP.
* **About:** Overview of the event’s mission, purpose, and background.
* **Gallery:** A visual archive showcasing artwork and performances from the events past and present.
* **Register:** Online RSVP form connected to Google Sheets API for real-time attendee tracking.
* **Chessed:** Highlights the charitable organization supported by the event.
* **Contact:** Provides an easy way for users to reach out with questions.
* **Mobile-Friendly Navigation:** Fully responsive design for seamless browsing on any device.

---

## Technologies Used
* Frontend: HTML, CSS (Bootstrap), JavaScript
* Backend: Python (Flask)
* Data Handling: Google Sheets API for RSVP submissions
* Hosting: Render (Auto-deployed on commits)
